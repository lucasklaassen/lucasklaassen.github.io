**Internal integration for LMK Management Inc. Not a public product.**

## Overview

LMK QBO Integration is an internal tool used only by Lucas Klaassen / LMK Management Inc. It connects to QuickBooks Online via Intuit's official API for internal bookkeeping and reporting.

This is not a public app. There are no external users, no sign-up flow, and no hosted web application.

## Disconnect

To revoke access, go to Intuit → Settings → Connected Apps and disconnect this integration. You will be redirected to the [disconnect notice](/legal/qbo-integration/disconnected/) page.
