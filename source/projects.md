## MMA Fantasy

[Visit MMA Fantasy](https://www.mma-fantasy.com)

A full-stack mobile application that brings fantasy sports to mixed martial arts. Built with React Native (Expo), Node.js, and PostgreSQL, MMA Fantasy serves **2,000+ monthly active users** who compete in fantasy leagues for UFC events.

**Key Features:**
- Real-time scoring during live UFC events
- Custom league creation and management
- Live chat and social features
- Push notifications for fight updates
- Comprehensive fighter statistics and analytics

**Tech Stack:**
- **Frontend:** React Native (Expo) for cross-platform iOS and Android
- **Backend:** Node.js serverless lambdas on AWS
- **Database:** PostgreSQL
- **Real-time:** WebSocket server for live updates
- **Infrastructure:** AWS (Lambda, RDS, API Gateway, S3)

MMA Fantasy serves as my laboratory for AI-native development practices, where I dogfood the latest tools and workflows before bringing them to client engagements.
